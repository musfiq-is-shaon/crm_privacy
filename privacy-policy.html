By using the App, you agree to the collection and use of information in accordance with this Privacy Policy.

1. Information We Collect

We may collect the following types of information when you use the App.

When you create an account or use authentication features, we may collect:

  • Name
  • Email address
  • Phone number
  • Company name
  • Job role or position
  • User profile information

1.2 Business Data

As a CRM system, the app stores business-related information including:

  • Customer contacts
  • Companies

  • Tasks and activities
  • Notes and logs
  • Expenses
  • Assigned users
  • Business communications

This data is entered by users and belongs to the organization using the system.

1.3 Technical Information

We may automatically collect certain technical information such as:

  • Device type

  • Log data

1.4 Authentication Data

To provide secure access to the system, the app uses authentication tokens (such as JWT tokens) which are securely stored on the device.

2. How We Use Your Information

We use the collected information to:

  • Provide and operate the CRM application
  • Authenticate and manage user accounts
  • Synchronize data with backend APIs
  • Manage contacts, deals, companies, and tasks
  • Improve performance and reliability
  • Provide customer support
  • Send important system notifications
  • Detect and prevent security issues or fraud

3. Data Storage and Security

We take appropriate measures to protect your information.

Security measures include:

  • Secure API communication (HTTPS)
  • Encrypted token storage using secure storage
  • Authentication using secure JWT tokens
  • Access control for user roles
  • Backend security protections

However, no method of transmission over the Internet is completely secure.

https://be-crm-production-a948.up.railway.app
